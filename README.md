# Simple CRUD Operation

A simple CRUD (Create, Read, Update, Delete) application built for learning and practice purposes.

This project demonstrates basic CRUD functionality using Laravel and MySQL.

---

## Features

- Create new records
- View all records
- Update existing records
- Delete records
- Form validation
- Clean and simple UI

---

## Tech Stack

- PHP
- Laravel
- MySQL
- Blade Template
- Bootstrap (optional)

---

## Installation

Follow the steps below to run the project locally:

##bash
git clone https://github.com/your-username/simple-crud-operation.git
cd simple-crud-operation
composer install
cp .env.example .env
php artisan key:generate


##Usage
Open browser and go to:
http://127.0.0.1:8000
Add, edit, view, and delete records using the UI
